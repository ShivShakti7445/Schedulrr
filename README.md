## 🤖 Introduction

**Schedulrr** is a modern scheduling platform that streamlines booking meetings without back-and-forth emails. Users can create event types, define weekly availability, share public booking links, and automatically generate Google Calendar events with Google Meet links. It offers a polished booking experience, protected routes, and a creator dashboard with upcoming meetings.

---

## ⚙️ Tech Stack

💻 **Frontend:** Next.js 14 (App Router, Server Components, Server Actions), React 18  
🛠️ **Backend:** Prisma ORM, Node.js  
🗄️ **Database:** Neon (PostgreSQL)  
🔐 **Auth:** Clerk (Google OAuth)  
📅 **Calendar API:** Google Calendar (Meet Link Integration)  
🎨 **UI:** Shadcn UI (Radix + TailwindCSS)  

---

## ✨ Features

✅ **Authentication & Authorization (Clerk)** – Secure sign-in/sign-up, protected routes via middleware  
✅ **User Onboarding** – Auto-create user record synced with Clerk profile  
✅ **Event Types** – Create customizable event types (title, description, duration, privacy)  
✅ **Public Booking Pages** – Shareable booking links like `/[username]/[eventId]`  
✅ **Availability Management** – Configure day-wise availability & minimum time gaps  
✅ **Smart Time Slots** – Auto-generate available slots for next 30 days, avoiding overlaps  
✅ **Google Calendar Integration** – Creates events with attendees + Google Meet link  
✅ **Booking Management** – Store bookings with details + cancel events with `googleEventId`  
✅ **Dashboard Overview** – Displays upcoming meetings for quick visibility  
✅ **Responsive UI/UX** – Smooth, mobile-first, accessible  
✅ **Reusable Components** – Powered by Shadcn UI & Tailwind  
✅ **Form Validation** – Zod + React Hook Form for strong validation  
✅ **Secure Server Actions** – All mutations protected with Clerk + `"use server"`
